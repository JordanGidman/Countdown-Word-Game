RULES OF THE GAME

This is a word game inspired by the TV show countdown. The aim is to create the longest word you can from a group of random letters that will be shown on screen.

There are 5 rounds per run. Once you click start the timer will begin counting down from 30. Look at the assortment of letters then type in the longest word you can think of. Then type the word into the box and press submit. after 5 rounds of this the score is added up

1. The word must be a real word

2. The word must be made only with the letters shown on screen

3. If time runs out before you click the submit button you get 0 points for that round.

4. Press start after each submit to start the next round.
